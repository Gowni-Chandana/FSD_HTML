Understanding File Paths in HTML

1. File Paths in HTML

File paths help in locating files like images, stylesheets, and other resources.

Absolute Path Example:

<img src="https://www.example.com/images/photo.jpg" alt="Image">

Relative Path Example:

<img src="images/photo.jpg" alt="Image">
